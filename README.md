# lists

Listas que Lean Browser descarga en runtime.

`sso_hosts.txt` son los dominios de login que el navegador no debe mover entre contextos, para que un login nunca se rompa a mitad del flujo. Lean la baja solo, aca no hay nada que instalar ni configurar.

Si un login te falla en Lean, abre un issue con el dominio y que proveedor es.
